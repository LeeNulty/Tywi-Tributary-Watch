# Tywi Tributary Watch

Independent, exploratory observations of the small rivers feeding the Tywi.

## About

These reports record straightforward field observations, including kick sampling, habitat condition and chemical results where available. They are intended to make local river evidence easy to find, discuss and build on.

**Important:** these are independent exploratory surveys, not statutory water-quality classifications. Results are site- and date-specific, and should be interpreted alongside weather, flow and habitat conditions.

## Publish it

1. Create a free GitHub account if you do not have one.
2. Create a new public repository named `tywi-tributary-watch`.
3. Upload the contents of this folder (not the folder itself) to the repository.
4. In the repository settings, open **Pages**, select **Deploy from a branch**, choose `main` and the root folder, then save.
5. The site will receive a public `github.io` address. Add that address to Google Search Console and request indexing for the home page and each report page.

## Add a future report

Copy `reports/cennen-2026-04-10.html`, change the title, date, description and findings, then add a new card linking to it from `index.html`.

Use descriptive file names such as `reports/dulais-2026-07-23.html`. Keep the report page's title and description specific to the river, location and date so search engines can distinguish it.

## Displaying results

Add a small results table near the top of every report. Include the sample duration, date and site, indicator groups observed, key organisms absent or present, and chemical readings such as temperature, pH, conductivity, nitrate, phosphate and dissolved oxygen where measured. Always include units, and use `not measured` rather than leaving a field blank.
